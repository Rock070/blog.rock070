如何切換 Git 遠端分支

2022年6月3日 Git 2 分钟 讀完 (約 227 字)

檢查對應的遠端分支

git branch -vv

更換分支

  • 方法一

git branch -u 遠端庫名/分支名

[!NOTE] 什麼是 HEAD
目前所在的分支。
可參考：【冷知識】HEAD 是什麼東西？

  • 方法二

git push -u 遠端庫名/分支名

同時會把 commit 推向遠端
